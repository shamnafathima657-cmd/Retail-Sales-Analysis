# Retail-Sales-Analysis
Retail Sales Analysis project using Excel, SQL, and Power BI
# Retail Sales Analysis Dashboard
## Project Overview
This project focuses on analyzing retail sales data to identify key business insights such as total sales, profit trends, top-performing products, and regional sales performance.
The analysis was performed using Excel for data cleaning, MySQL for querying and data analysis, and Power BI for building an interactive dashboard.

## Objectives
* Analyze total sales and profit performance
* Identify monthly sales trends
* Find top-selling products
* Analyze sales and profit by category
* Understand regional sales distribution

## Tools Used
* Excel – Data cleaning and preparation
* MySQL – Data analysis using SQL queries
* Power BI – Interactive dashboard visualization

## Dataset
The project uses a retail dataset containing information such as:

* Order ID
* Order Date
* Product Name
* Category
* Sales
* Profit
* Region
* Customer Name

## SQL Analysis Performed
Some of the key SQL analyses performed in this project include:

* Total Sales
* Total Profit
* Total Orders
* Monthly Sales Trend
* Sales by Category
* Profit by Category
* Top Products by Sales
* Sales by Region
* Top Customers

## Dashboard
An interactive dashboard was created using Power BI to visualize key business metrics including:

* Total Sales
* Total Profit
* Monthly Sales Trend
* Sales by Category
* Profit by Category
* Top Products
* Sales by Region

## Key Insights
* Technology category generated the highest profit
* Certain months showed higher sales trends compared to others
* A small group of products contributed significantly to overall sales
* Sales distribution varied across different regions

## Conclusion
This project analyzes retail sales data to identify key business insights such as sales trends, top products, and profit by category. Using Excel, SQL, and Power BI, the data was cleaned, analyzed, and visualized to support better business decision-making.
